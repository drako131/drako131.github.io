Complete Guide: Run Valorant in a Virtual Machine with Hyper-V

VM METHOD

⚠️ IMPORTANT REQUIREMENTS

✓ Windows 10 or Windows 11 Pro Edition (activation guide included below)

✓ Hyper-V capable hardware (most modern CPUs support this)

✓ At least 16GB RAM recommended

✓ Dedicated GPU recommended for best performance

📝 Step 1: Activate Windows 10/11 Pro

If you don't have Windows Pro activated, follow these steps to activate it for free using MAS (Microsoft Activation Scripts).

1 Open PowerShell as Administrator

Right-click the Start button and select "Windows PowerShell (Admin)" or "Terminal (Admin)"

2 Run the activation command

Copy and paste this command into PowerShell:

irm https://get.activated.win | iex
3 Follow the activation wizard

✓ Wait for a new command prompt window to open

✓ Press 1 for Windows activation

✓ You should see: "Windows 10 Pro is permanently activated with a digital license."

ℹ️ Alternative Command (if blocked by ISP/DNS)

iex (curl.exe -s --doh-url https://1.1.1.1/dns-query https://get.activated.win | Out-String)

⚙️ Step 2: Enable Hyper-V

1 Open Windows Features

Press Win + R, type optionalfeatures and press Enter

2 Enable required features

Check the following boxes:

  • Hyper-V (all sub-options)
  • Virtual Machine Platform
  • Windows Subsystem for Linux
3 Apply and restart

Click OK and restart your PC when prompted

🌐 Step 3: Create Virtual Network Switch

1 Open Hyper-V Manager

Search for "Hyper-V Manager" in Start menu

2 Create Virtual Switch

✓ Click "Virtual Switch Manager..." on the right panel

✓ Select "External" and click "Create Virtual Switch"

✓ Give it a name (e.g., "VM-Network")

3 Configure network adapter

✓ Select your primary Network Controller (e.g., Intel Ethernet Controller)

✓ ⚠️ IMPORTANT: Check "Allow management operating system to share this network adapter"

✓ Click OK to apply

💾 Step 4: Download GPU Partition Scripts

1 Download the GPU-P scripts

Download the official GPU partitioning scripts from GitHub. The package contains:

  • New-GPUPDriverPackage.ps1
  • New-GPUPVirtualMachine.ps1
  • readme.md
📥 Download GPU-P Scripts (ZIP)
2 Extract the files

Extract the downloaded ZIP file to a location you can easily access (e.g., C:\VM-Scripts)

3 (Optional) Customize VM settings

Open New-GPUPVirtualMachine.ps1 with Notepad or Visual Studio Code to modify VM parameters like RAM, CPU cores, etc.

✅ Bypass HWID Bans

If your main PC is HWID banned, you can use this VM method as a "2nd PC" to play on your main account:

  • Create the VM following this guide
  • If the VM gets banned, simply delete it and create a new one
  • Start from Step 5 (you don't need to redo Hyper-V setup)
  • You'll have a "fresh" system each time without hardware bans affecting your host PC

Nếu bạn chưa kích hoạt Windows Pro, hãy làm theo các bước sau để kích hoạt miễn phí bằng MAS (Microsoft Activation Scripts).

Nhấp chuột phải vào nút Start và chọn "Windows PowerShell (Admin)" hoặc "Terminal (Admin)"

irm https://get.activated.win | iex
3 Làm theo trình hướng dẫn kích hoạt

✓ Đợi cửa sổ command prompt mới mở ra

🖥️ Bước 5: Tạo Virtual Machine

1 Mở PowerShell với quyền Administrator

Nhấp chuột phải Start → Windows PowerShell (Admin)

2 Đặt execution policy
Set-ExecutionPolicy Bypass

Xác nhận bằng cách gõ Y và nhấn Enter

3 Di chuyển đến thư mục scripts
cd C:\VM-Scripts

Thay thế bằng đường dẫn thực tế của bạn. Xác minh bằng lệnh ls.

4 Thực thi script tạo VM
.\New-GPUPVirtualMachine.ps1

Đợi script hoàn thành. Bạn sẽ thấy VM mới tên "GPU-VM" trong Hyper-V Manager.

⚙️ Bước 6: Cấu Hình Virtual Machine

Trong Hyper-V Manager, nhấp chuột phải "GPU-VM" → Settings

2 Điều chỉnh Memory và Processor

✓ Memory: Phân bổ ít nhất 8GB (8192 MB) hoặc nhiều hơn

✓ Processor: Gán ít nhất 4 cores

3 Gắn Network Adapter

✓ Đi đến phần "Network Adapter"

✓ Chọn Virtual Switch đã tạo trước đó

✓ Nhấp Apply

💿 Bước 7: Cài Windows 10 trong VM

1 Tải Windows 10 ISO

Tải từ trang web chính thức của Microsoft

2 Thêm ISO vào VM

Trong VM Settings:

  • Đi đến "SCSI Controller"
  • Chọn "DVD Drive" → Add
  • Chọn "Image file" và duyệt đến file Windows 10 ISO của bạn
3 Tạo Virtual Hard Drive

Vẫn trong "SCSI Controller" settings:

  • Chọn "Hard Drive" → Add → New
  • Chọn "Dynamically expanding" hoặc "Fixed size"
  • Đặt kích thước ít nhất 80GB (khuyến nghị: 120GB+)
  • Ở bước 4, chọn "Create a new blank virtual hard disk"
  • Hoàn thành wizard
4 Đặt Boot Order

Đi đến phần "Firmware" và sắp xếp thứ tự khởi động:

  1. DVD Drive (Windows 10 ISO)
  2. Hard Drive

Nhấp Apply → OK

❓ Vấn Đề Thường Gặp & FAQ

H: Tôi gặp lỗi kết nối video khi kết nối VM

Đ: Tiếp tục nhấp "Reconnect" nhiều lần. Đây là vấn đề Hyper-V phổ biến thường được giải quyết sau vài lần thử.

H: Valorant crash vài phút sau khi khởi động

Đ: Đảm bảo Cloudflare WARP đã bật. Cũng xác minh rằng:

  • TPM KHÔNG được bật trong cài đặt VM
  • Bạn đã bật "Virtual Machine Platform" trong Windows features (bên trong VM)
  • Secure Boot, TPM, và HVCI được cấu hình đúng
H: Tôi thấy lỗi "A D3D11-compatible GPU is required"

Đ: Điều này có nghĩa là phân vùng GPU không hoạt động đúng. Xác minh:

  • Gói driver GPU đã được giải nén đúng vào C:\Windows
  • Bạn đã chạy lệnh Set-VMProcessor
  • GPU của bạn hỗ trợ phân vùng (hầu hết GPU hiện đại đều có)
H: VM rất lag hoặc chậm

Đ: Phân bổ thêm tài nguyên:

  • Tăng RAM lên 12GB+ nếu có thể
  • Gán thêm CPU cores (khuyến nghị 6-8)
  • Đảm bảo PC chính không chạy các chương trình tốn nhiều tài nguyên

✅ Vượt Qua HWID Ban

Nếu PC chính của bạn bị HWID ban, bạn có thể dùng phương pháp VM này như "PC thứ 2" để chơi tài khoản chính:

  • Tạo VM theo hướng dẫn này
  • Nếu VM bị ban, chỉ cần xóa nó và tạo cái mới
  • Bắt đầu từ Bước 5 (không cần làm lại thiết lập Hyper-V)
  • Bạn sẽ có hệ thống "mới" mỗi lần mà không ảnh hưởng đến PC chính
